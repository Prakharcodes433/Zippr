Introduction

Welcome to the Zippr documentation page. Here you will find all the information you need to get started and use Zippr efficiently.

Installation

To install Zippr, simply download the latest release and include it in your project:

<script src="zippr.min.js"></script>

Usage

Initialize Zippr in your JavaScript code:

const zip = new Zippr();
zip.compress(files);
zip.extract(archive);

API Reference

  • Zippr.compress(files) - Compresses an array of files.
  • Zippr.extract(archive) - Extracts files from an archive.

FAQ

Q: Is Zippr free to use?
A: Yes, Zippr is open source and free for all uses.

Q: What file types are supported?
A: Most common file types are supported for compression and extraction.
